# 페르소나 탐색 — 나를 찾아가는 기록

> 정리되지 않은 생각을 꺼내놓고, 대화를 통해 "나는 누구인가"를 찾아가는 누적 기록.

---

## 규칙

1. **방향은 내가 정한다.** AI는 질문·정리·자료 수집만 돕는다.
2. **거짓말만 쓰지 않는다.** 정직이 유일한 규칙이다.
3. **이전 답은 지우지 않는다.** 생각이 바뀌어도 이전 답은 남긴다. 변화의 궤적이 곧 나다.
4. **추상어로 도망치지 않는다.** 항상 구체 사례로 내려온다.
5. **한 번에 끝내지 않는다.** 살면서 갱신한다.

## 이 작업이 왜 단계적으로 이루어지는가

자기를 "만든다"가 아니라 "발견한다".
답을 미리 정해놓고 채우는 게 아니라,
정직한 답들이 쌓여 스스로 형태를 드러내는 것.

그래서 순서는 이렇게 간다:

```
세계관 (이미 있는 것)
  ↓
질문에 정직히 답한다
  ↓
과거의 사상가들과 비교한다
  ↓
반복되는 패턴·모순이 보인다
  ↓
그 패턴을 정리하면 → 나의 선언문이 된다
  ↓
그 선언문을 실현하면 → 시스템이 된다
```
